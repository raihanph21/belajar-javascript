Variabel

Deklarasi

Mendaftarkan variabel ke dalam lingkup yang sesuai

Inisialisasi

menyediakan memori untuk variabel

Assignment

menetapkan nilai yang spesifik ke dalam variabel

Contoh

var x; -> Deklarasi dan Inisialisasi

x = 20; -> assignment

note penting

biasakan selalu deklarasi dan Inisialisasi sebelum assignment

variabel shorthand

var nama = "raihan", umur = "21", kelamin = "laki-laki";
