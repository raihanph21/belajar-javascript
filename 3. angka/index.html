Angka pada javascript

  • Angka tanpa desimal = 10, 15, 123456 (akurat sampai 15 digit)
  • angka desimal = 3.14, 2.90
  • eskponen = 123e5 -> 12300000, 123e-5 -> 0.00123
  • bilangan negatif = -25, -0.25
  • angka spesial = 2/0 -> infinity, -2/0 -> -infinity, 0/0 -> NaN (not a number) sama kaya angka dibagi string 100/"apel"

100/"10" akan dianggap 100/10 (membaca string menjadi angka)

maksimal 17 digit belakang koma

jangan mengawali angka pada js dengan 0, seperti 022 (akan dianggap bilangan oktal)

jangan mengawali angka dengan 0x (akan dianggap bilangn hexadesimal)
